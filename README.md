# vinyl-github-src
use github repos as a vinyl (gulp) source

## usage
```js
var vinylGithubSrc = require('vinyl-github-src')
```


## api


## installation

    $ npm install vinyl-github-src


## running the tests

From package root:

    $ npm install
    $ npm test
